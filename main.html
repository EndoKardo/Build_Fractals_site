Программа для построения фракталов

Ручная документация

Зачем это?

Данное приложение было разработано для тех, кому необходимо построить фрактал.
Для работы с программой необходимо просто скачать приложение.
Пройдя регистрацию, вы сможете воспользоваться всеми функциями приложения.

Что есть?

Данное приложенеи включает в себя такие модули-компоненты:

  • 1. Построение фракталов
  • 2. Настройка построения фракталов
  • 3. Выбор реализации алгоритмов фрактального построения:
    a. Алгоритмическое построение;
    b. Геометрическое построение;
    c. Построение на основе метода IFS.
  • 4. Тайм-трекинг:
    a. Отслеживание времени выполнения построения;
    b. Отслеживание текущего времени.
  • 5. Копирование данных для предотвращения их потери:
    a. Резервное (на компьютер);
    b. На облако.
  • 6. Чат-бот:
    a. Для быстрого взаимодействия с проектом;
    b. Для нахождения ответов по функционалу проекта.
  • 7. Шаблоны фракталов:
    a. Стандартные шаблоны;
    b. Новые шаблоны (от пользователей);
    c. С нуля шаблон.
  • 8. Связь с разработчикам:
    a. Для предложения какого-либо нового нововведения;
    b. Для прямой связи с разработчиками по разным вопросам.
  • 9. Реальное время в облаке
  • 10. Социальное взаимодействие:
    a. Обмен опытом в проекте;
    b. Обмен задачами между пользователями внутри платформы;
    c. Контактирование с продвинутыми пользователями этого приложения с целью задать вопросы по разным вопросам.
